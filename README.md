# MJProject
## 작성일 : 2022-02-07 작성자 : 장나현

## 프로젝트 명 : MJ (Mbti Journey, mbti 테스트를 통한 랜선 여행 사이트)

## 기획 의도
```
- 많은 사람들에게 재미있는 요소이자 자기소개에 필수 요소가 되고 있는 mbti를 통해, 그에 관한 여행 테스트를 진행하여 다양한 여행지에 대한 여러 정보를 나누는 사이트

- 코로나가 잠잠해지지 않고 더욱 심해지면서 유투브 동영상을 통해 또 나의 갤러리를 통해 과거의 여행을 되새기는 것이 조금이나마 위로가 되었습니다.
그러나 다시금 여행에 대해 더욱 갈망하게 만들었습니다.
그리하여 요즘 유행하는 mbti 테스트를 기반으로 랜선으로 여행을 떠날 수 있는 사이트를 만들고자 계획하게 되었습니다.

- 초기 기능 기획 (대표적인 기능)
1. 임의로 자체 제작한 여행지 mbti 테스트 (마이페이지에 테스트 결과 보여주기)
2. 여행루트를 작성하여 업로드하는 게시판 (구글맵 API 사용, 지도에 마커로 표시하여 연결)
3. 여행루트를 찜하여 마이페이지에 보여주기
4. 여행 사진을 업로드하고 좋아요를 누르는 사진 전용 게시판
```

## 벤치마킹
```
1. instagram 게시물 구성, 좋아요, 댓글 기능
  - 인스타그램 게시물 구성과 비슷한 카드를 사진 전용 게시판에 나열하고 좋아요와 댓글 기능을 모달창으로 만들어 사진 게시판 구성하고자 함.
  
2. velog 해시태그 기능
  - 벨로그의 해시태그 기능을 참고해 해시태그로 게시물을 분류하고자 함.
```

## 주요 기능
```
1. 로그인 및 회원가입
  - 일반, 카카오 로그인 API
  
2. mbti 테스트
  - 임의의 자체제작 mbti 테스트
  - 테스트 후 회원 mbti 정보 저장
  - mbti 테스트 결과에 대한 여행지 추천 글 보여주기
  
3. 사진 전용 게시판
  - 회원이 여행 사진을 업로드하고 좋아요 및 댓글을 작성하는 게시판
  - 모달창 이용
  - 무한 스크롤
  
4. 구글맵 API를 사용한 여행루트 게시판
  - 회원이 각 여행지에 대한 여행루트를 소개 및 공유하는 게시판
  - 구글맵에 마커를 찍고 연결하여 여행루트를 보여줌.
  - 해시태그 기능 (제한글자 : 여행지, mbti만 가능)
  - 무한 스크롤
  
5. 오픈 채팅방
  - mbti 또는 여행지에 대한 카테고리로 오픈 채팅방을 개설해 1:1, 1:N으로 채팅
  
6. 관리자 페이지
  - 게시물 삭제 및 회원 관리
  
7. 마이페이지
  - 테스트를 통한 mbti 정보 보여주기
  - 여행루트 게시판에서 찜한 여행루트 보여주기
  - 작성한 게시글 수정
  - 회원 정보 수정
```
